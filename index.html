Questions And Answers

JavaScript is a single threaded programming language and also a dynamic types of programming language. It was designed to control the behavior of the website at the client level that is browser. It can handle one task at a time and every line of code is executed line by line. Also it allocates the memory to all functions and variables in the program. JavaScript execute the complex actions and also enable interactions of website with users.

The difference between local storage and session storage is local storage store data for long time. Even if the browser tab is close and after long time if you reopen the same tab in browser, you will get the same data from local storage.
Session storage is r run time process. It store the dada in execution time. If you close the tab you will loss the session storage data.
